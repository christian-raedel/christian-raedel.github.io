CQModel

new CQModel(opts, data)

CQModel - A collection class for using with CStore. All methods returning a value, return a promise for that value. Thereby its an asynchron version of CModel.
Parameters:
Name Type Description
opts object (optional) - An object holding options for creating a new CQModel instance. At the moment, the only valid option is a 'name' for the collection.
data array (optional) - An array of objects, representing the collection.

(static) find(query) → {array}

Find all documents matching a simple field/value comparation or a complexer query in the form of '{field: {'$comparator': 'value'}}'. At the moment, the comparator functions '$eq' (equal), '$ne' (not equal), '$gt' (greater than), '$lt' (less than) and '$in' (member of a list of items) are implemented. A special comparator is the '$or' one, which is looking for a field/value OR another field/value.
Parameters:
Name Type Description
query object A search field or a query.
Source:
  • cqmodel.js, line 56
Returns:
- The found document or an empty array.
Type
array
Example
var CStore = require('node-cstore');
var model = new CStore.CQModel();

model.insert({dress: '$noir', material: 'silk'});
var items = model.find({dress: '$noir'}); //simple field/value

var items = model.find({dress: {'$eq': '$noir'}}); //simple query
var items = model.find({dress: {'$eq': '$noir'}, material: {'$eq': 'silk'}}); //AND
var items = model.find({'$or': {dress: {'$eq': '$noir'}, material: {'$eq': 'silk'}}}); //OR
