Class: CSModel

CSModel

new CSModel(opts, data)

CSModel - A collection class for using with CStore. All methods are inherited directly from CModel.
Parameters:
Name Type Description
opts object (optional) - An object holding options for creating a new CSModel instance. At the moment, the only valid option is a 'name' for the collection.
data array (optional) - An array of objects, representing the collection.

Example
var CStore = require('node-cstore');
var model = new CStore.CSModel({name: 'MyCollection'});
