Class: CStore

CStore

new CStore(opts, models)

CStore - A document store.
Parameters:
Name Type Description
opts object (optional) - Arguments for creating a new CStore instance. Valid options are a 'name', a 'filename' for persisting the document store and a CLogger instance. If filename is omitted, the datastore is inmemory only. With no given logger, the constructor will create a new one.
models array (optional) - A list with CModel instances for holding the data.

Example
var CStore = require('node-cstore');
var store = new CStore({name: '$apple'}, {
     'mac-books': new CStore.CModel()
});
