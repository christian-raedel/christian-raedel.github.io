Wampify

A node.js WAMP-style RPC & Publish/Subscribe implementation using WebSockets. RPC functions can be plugged-in from ordinary node-modules.

Installation

npm install --save git+https://github.com/christian-raedel/wampify.git

Testing

cd /path/to/wampify
make install test

Documentation

cd /path/to/wampify
make install docs
$BROWSER doc/index.html

or browse online
